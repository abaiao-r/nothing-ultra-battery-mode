# Ultra Save Nothing

Xiaomi-style ultra battery mode app for Nothing Phone (3a Pro). Ships with a no-root feature set
(distraction-lock launcher shell, allowlist, silent mode) plus a real, root-powered mode — Battery
Saver enforcement, Wi-Fi/mobile data and dark-theme control, per-core CPU frequency capping, and
periodic force-stopping of any app not on the allowlist — that runs automatically when the device
has root (`su`) access, and safely no-ops otherwise. See `internal_docs/project-decisions.md`
(local-only) for the full rationale behind the root pivot.

See `CONTRIBUTING.md` and `docs/governance.md` for branch rules, DoD, ATP policy, and architecture contracts.
